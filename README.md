#STM32F103C8T6 智能小车项目 (by 404elf)

## 项目简介

本项目是一款基于 **STM32F103C8T6** 的入门级智能小车。
开发环境为：Keil uVision5，调试器选择为ST-Link
代码**源自B站UP主“好家伙VCC”的教程视频**：[STM32智能小车V1.5](https://www.bilibili.com/video/BV1SY411L7rJ/)。在此基础上，**利用时间戳和状态机重构了业务逻辑**，并**参考B站UP主“江协”的模块化思想优化了文件架构与代码管理**。

因为部分引脚的损坏，所以在软件程序和硬件连接上进行了修改，并不能根据教程视频硬件连接直接烧录使用。
由于时间久远大概只记得是“超声波模块“和“蓝牙模块“的开发板相应引脚损坏，在使用前请注意仔细核对。
与超声波ECHO引脚连接的芯片引脚无法长期忍耐5V高压所以损坏，可以改连超声波模块的3.3V引脚。
芯片与蓝牙模块连接的rx，tx引脚产生损坏，所以对其进行了修改。

## 功能模式

小车支持以下操作模式：

*   **MODE0: 停止**
*   **MODE1: 超声波跟随**
*   **MODE2: 蓝牙遥控**
*   **MODE3: 超声波避障**
*   **MODE4: 普通循迹**
*   **MODE5: PID循迹** (当前PID参数待调试)

## 文件架构

项目采用分层设计：

*   **Start/ ：** 启动文件。
*   **Library/ ：** 标准库函数。
*   **Algorithm/ ：** 算法层，目前主要包含PID算法。
*   **Hardware/ ：** 硬件层，包含各硬件的驱动。
*   **System/ ：** 系统层，负责全局宏定义与变量。
*   **User/ ：** 执行层，包含初始化与业务逻辑。
